## Introduction

A differential amplifier amplifies the difference between two input signals while rejecting common signals. This makes it very useful in noise reduction and signal conditioning.

MOS-based differential amplifiers are widely used in analog ICs such as op-amps and comparators due to their high gain and efficiency.

---

## Theory

The circuit uses two identical MOSFETs with a shared current source.

Differential input:
v_id = v_in1 - v_in2  

- If v_in1 > v_in2 → M1 current increases  
- If v_in2 > v_in1 → M2 current increases  

For small inputs → linear operation  
For large inputs → non-linear behavior  

Gain:
A_v = g_m × R_out  

Transconductance:
g_m = (2 I_D) / V_ov  

---

## Design Calculations

### Given

VDD = 0.9V  
VSS = -0.9V  
Power ≤ 1.8mW  
L = 480nm  
VT = 0.36V  
Vp = -0.7V  

---

### Power Calculation

P = (VDD - VSS) × ISS  

= (0.9 - (-0.9)) × ISS  
= 1.8 × ISS  

Given P ≤ 1.8mW  

1.8 × ISS ≤ 1.8 × 10⁻³  

ISS ≤ 1mA  

Choose: ISS = 1mA  

---

### Drain Current

ID1 = ID2 = ISS / 2  

= 1mA / 2  
= 0.5mA  

---

### Load Resistance

Vout = VDD - ID × RD  

0 = 0.9 - (0.5 × 10⁻³ × RD)  

RD = 0.9 / (0.5 × 10⁻³)  

RD = 1.8kΩ  

---

### Bias Calculation

VG = 0V  
VS = -0.7V  

VGS = VG - VS  
= 0 - (-0.7)  
= 0.7V  

VOV = VGS - VT  
= 0.7 - 0.36  
= 0.34V  

VDS = VD - VS  
= 0 - (-0.7)  
= 0.7V  

Check: VDS > VOV → 0.7 > 0.34 ✔  

---

### Width Calculation

Formula:

ID = (1/2) μnCox (W/L) (VOV²)  

Rearrange:

W = (2 ID L) / (μnCox × VOV²)  

Substitute:

ID = 0.5 × 10⁻³  
L = 480 × 10⁻⁹  
μnCox = 2.365 × 10⁻⁴  
VOV = 0.34  

W = (2 × 0.5 × 10⁻³ × 480 × 10⁻⁹) / (2.365 × 10⁻⁴ × 0.34²)  

W ≈ 17.57 µm  

Final (after tuning): ≈ 28.475 µm

  # Differential Amplifier (PMOS Load + NMOS Tail Source)

## 1. Overview of Operation

This configuration uses:

- NMOS pair → M1, M2 (input stage)
- NMOS → M5 (tail current source)
- PMOS → M3, M4 (active load mirror)

The differential signal is:

v_id = v_in1 - v_in2

### Current Steering Behavior

- v_in1 ↑ → current shifts to M1  
- v_in2 ↑ → current shifts to M2  

The total current (ISS) from M5 is redistributed between the two branches.

### Role of PMOS Load

- M3 → diode-connected reference  
- M4 → mirrored branch  

This arrangement gives **high output resistance**, improving gain.

### Signal Behavior

- Small input → both transistors ON → linear output  
- Large input → one OFF → distortion

## Circuit Overview

The amplifier includes:

- NMOS input pair → M1, M2  
- PMOS current mirror → M3, M4  
- NMOS current source → M5  

The circuit is biased symmetrically and driven by differential inputs.

---

## Given Data

### Supply
- VDD = 0.9 V  
- VSS = -0.35 V  

### Input
- Vin1 = SINE(0, 10 mV, 1 kHz)  
- Vin2 = SINE(0, -10 mV, 1 kHz)  

So,

Vid = Vin1 - Vin2 = 20 mV

- Threshold voltage ≈ 0.366 V  

---

## Saturation Conditions

NMOS:
VDS ≥ VGS − VT  

PMOS:
VSD ≥ VSG − |VT|  

---

## Region Check (DC Analysis)

---

### M1 (NMOS)

VGS1 = 0 − (-0.7) = 0.7 V  

VDS1 = 0.3 − (-0.7) = 1.0 V  

Check:

VDS1 ≥ VGS1 − VT  

1.0 ≥ 0.7 − 0.366  

1.0 ≥ 0.334 ✔  

➡ M1 is in saturation  

---

### M2 (NMOS)

Same values as M1  

➡ Also in saturation  

---

### M3 (PMOS)

VSD3 = 0.9 − 0.3 = 0.6 V  

VSG3 = 0.9 − 0.3 = 0.6 V  

Check:

0.6 ≥ 0.6 − 0.366  

0.6 ≥ 0.234 ✔  

➡ Saturation  

---

### M4 (PMOS)

Same as M3  

➡ Saturation  

---

### M5 (NMOS)

VGS5 = 0.366 − (-0.35) = 0.716 V  

VDS5 = -0.7 − (-0.35) = -0.35 V  

|VDS5| = 0.35 V

0.35 ≥ 0.716 − 0.366  

0.35 ≥ 0.35 ✔  

➡ Edge of saturation
