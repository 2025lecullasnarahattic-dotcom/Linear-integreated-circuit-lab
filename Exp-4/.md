## Introduction

A differential amplifier amplifies the difference between two input signals while rejecting common signals. This makes it very useful in noise reduction and signal conditioning.

MOS-based differential amplifiers are widely used in analog ICs such as op-amps and comparators due to their high gain and efficiency.

---

## Theory

The circuit uses two identical MOSFETs with a shared current source.

Differential input:
v_id = v_in1 - v_in2  

- If v_in1 > v_in2 → M1 current increases  
- If v_in2 > v_in1 → M2 current increases  

For small inputs → linear operation  
For large inputs → non-linear behavior  

Gain:
A_v = g_m × R_out  

Transconductance:
g_m = (2 I_D) / V_ov

## Design Calculations

### Given

VDD = 0.9V  
VSS = -0.9V  
Power ≤ 1.8mW  
L = 480nm  
VT = 0.36V  
Vp = -0.7V  

---

### Power Calculation

P = (VDD - VSS) × ISS  

= (0.9 - (-0.9)) × ISS  
= 1.8 × ISS  

Given P ≤ 1.8mW  

1.8 × ISS ≤ 1.8 × 10⁻³  

ISS ≤ 1mA  

Choose: ISS = 1mA  

---

### Drain Current

ID1 = ID2 = ISS / 2  

= 1mA / 2  
= 0.5mA  

---

### Load Resistance

Vout = VDD - ID × RD  

0 = 0.9 - (0.5 × 10⁻³ × RD)  

RD = 0.9 / (0.5 × 10⁻³)  

RD = 1.8kΩ  

---

### Bias Calculation

VG = 0V  
VS = -0.7V  

VGS = VG - VS  
= 0 - (-0.7)  
= 0.7V  

VOV = VGS - VT  
= 0.7 - 0.36  
= 0.34V  

VDS = VD - VS  
= 0 - (-0.7)  
= 0.7V  

Check: VDS > VOV → 0.7 > 0.34 ✔  

---

### Width Calculation

Formula:

ID = (1/2) μnCox (W/L) (VOV²)  

Rearrange:

W = (2 ID L) / (μnCox × VOV²)  

Substitute:

ID = 0.5 × 10⁻³  
L = 480 × 10⁻⁹  
μnCox = 2.365 × 10⁻⁴  
VOV = 0.34  

W = (2 × 0.5 × 10⁻³ × 480 × 10⁻⁹) / (2.365 × 10⁻⁴ × 0.34²)  

W ≈ 17.57 µm  

Final (after tuning): ≈ 28.475 µm

## 1.3 Transient Analysis and Linearity Observation

The linear behavior of the differential amplifier is verified using transient analysis.

### Condition for Linearity

$$
|V_{id}| < \sqrt{2} V_{OV}
$$

$$
\sqrt{2} V_{OV} = 1.414 \times 0.34 = 0.48V
$$

### Case 1: Linear Region

Input applied:

$$
V_{id} = 100mV < 0.48V
$$

## Gain Calculation

### Simulated

Vin(pp) = 100mV  
Vout(pp) = 604mV  

Av = Vout / Vin  
= 604 / 100  
= 6.04  

Gain(dB) = 20 log(6.04)  
= 15.62 dB  

---

### Theoretical

ro = 1 / (λID)  
= 1 / (0.1 × 0.5×10⁻³)  
= 20kΩ  

ro_eff = 10kΩ  

Rout = RD || ro  
= 1.8k || 10k  
≈ 1.53kΩ  

gm = (2ID)/VOV  
= (2 × 0.5×10⁻³)/0.34  
≈ 2.94mS  

Av = gm × Rout  
= 2.94×10⁻³ × 1.53×10³  
≈ 4.5
